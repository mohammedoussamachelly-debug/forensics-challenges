# FIJIBI

**Category:** Digital Forensics 

## Description

During an investigation into a small online fraud group, investigators recovered two short voice notes from a private communication channel. The recordings sound ordinary at first, but analysts suspect they contain information that wasn't intended to be heard directly.

Your task is to analyze both audio files, extract the hidden content, and use it to recover the flag.

## Files Provided

- `vn_2026-02-22_1841.wav`
- `vn_2026-02-22_1844.wav`
